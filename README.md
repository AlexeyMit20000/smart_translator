# Проект приложение переводчик текстовой информации

Это приложение представляет собой переводчик текстовой информации, разработанный с использованием собственных моделей машинного обучения.

## Описание проекта

Основная цель проекта — продемонстрировать процесс обучения и применения моделей машинного перевода. Приложение использует две независимые модели:

*   Модель для перевода с английского на русский язык.
*   Модель для перевода с русского на английский язык.

Обучение моделей производится на готовом наборе данных сайта Kaggle.

Ссылка на набор данных: https://www.kaggle.com/datasets/hijest/englishrussian-dictionary-for-machine-translate

## Структура проекта

Ключевые файлы и их назначение:

*   `main.py`: Основной исполняемый файл приложения. Запускает интерфейс переводчика и использует обученные модели для обработки текста.
*   `en_ru_train.ipynb`: Jupyter-ноутбук, предназначенный для обучения модели перевода с английского на русский язык.
*   `ru_en_train.ipynb`: Jupyter-ноутбук, предназначенный для обучения модели перевода с русского на английский язык.
*   `layers.py`: Модуль, содержащий пользовательские слои нейронной сети. Импортируется в `main.py`, чтобы избежать дублирования кода при загрузке моделей.
*   `requirements.txt`: Список всех необходимых библиотек и их версий для установки зависимостей проекта.

## Подготовка к запуску

Перед первым запуском приложения или процедурой обучения моделей необходимо создать требуемую структуру папок и разместить файлы с данными.

Обязательные шаги перед запуском:

1.  Создайте папку `dataset` в корневой директории проекта.
2.  Поместите файл с набором данных `rus.txt` в созданную папку `dataset`. Этот файл используется для обучения обеих моделей.
3.  Создайте папку `model` в корневой директории проекта. В этой папке будут хранится файлы обученных моделей `(*.h5)` после выполнения соответствующих ноутбуков.

```
Структура проекта:
├── main.py
├── layers.py
├── requirements.txt
├── en_ru_train.ipynb
├── ru_en_train.ipynb
├── /dataset
│ └── rus.txt
└── /model
└── (сюда положить файлы обученных моделей)
```

## Использование

Обучение моделей лучше производить в Google Colab:

1.  Откройте и запустите ячейки в ноутбуке `en_ru_train.ipynb` для обучения первой модели.
2.  Откройте и запустите ячейки в ноутбуке `ru_en_train.ipynb` для обучения второй модели.
3.  Убедитесь, что файлы моделей сохранились в папку `model`

### Запуск приложения

После успешного обучения и сохранения моделей можно запустить основное приложение `main.py`.
